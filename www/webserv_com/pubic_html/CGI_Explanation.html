CGI Explanation

About CGI

The Common Gateway Interface (CGI) is a standard protocol for interfacing external application software with an information server, such as an HTTP server. A CGI program is executed in real-time, so that it can output dynamic information.

CGI can be used to create various web applications. When a visitor opens a page built with CGI, the script will be executed on the server side. This execution can generate any sort of content, including images, video, audio or other binary data.
