Webserv - External functs

  • execve: Execute program
  • dup: Duplicate a file descriptor
  • dup2: Duplicate a file descriptor to a desired value
  • pipe: Create pipe
  • strerror: Return string describing error number
  • gai_strerror: Returns an error message for getaddrinfo
  • errno: Number of last error
  • fork: Create a child process
  • htons: Convert host byte order to network byte order (short)
  • htonl: Convert host byte order to network byte order (long)
  • ntohs: Convert network byte order to host byte order (short)
  • ntohl: Convert network byte order to host byte order (long)
  • select: Synchronous I/O multiplexing
  • poll: Wait for some event on a file descriptor
  • epoll_create: Creates an epoll instance
  • epoll_ctl: Control interface for an epoll descriptor
  • epoll_wait: Wait for an I/O event on an epoll file descriptor
  • kqueue: Provides a generic method of notifying the user when an event happens or a condition holds
  • kevent: Kernel event structure
  • socket: Create an endpoint for communication
  • accept: Accept a connection on a socket
  • listen: Listen for connections on a socket
  • send: Send a message on a socket
  • recv: Receive a message from a socket
  • bind: Bind a name to a socket
  • connect: Initiate a connection on a socket
  • getaddrinfo: Network address and service translation
  • freeaddrinfo: Free addrinfo structure
  • setsockopt: Set options on a socket
  • getsockname: Get socket name
  • getprotobyname: Get protocol number by name
  • fcntl: File control
  • close: Close a file descriptor
  • read: Read from a file descriptor
  • write: Write to a file descriptor
  • waitpid: Wait for process to change state
  • kill: Send signal to a process
  • signal: ANSI C signal handling
  • access: Check user's permissions for a file
  • opendir: Open a directory
  • readdir: Read a directory
  • closedir: Close a directory
