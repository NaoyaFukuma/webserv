Welcome to the Webserv HTTP Server Project!

About Webserv

Webserv is an HTTP server program built on top of the C++ programming language. With a focus on performance and reliability, Webserv provides an efficient and robust solution for serving web content.

Features

Our server supports a variety of features, including but not limited to, the handling of GET and POST requests, CGI scripts, automatic redirection, multithreading, and more.

Installation

To compile the project, clone the repository and run the 'make' command. The server executable will be generated and ready to use.
